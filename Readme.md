# Library System (Java)

Library management system developed in Java, focusing on collection organization and loan control.

## Features
- Book registration and management
- User registration and control
- Loan and return system
- Collection listing and organization

## Technologies
- Java
- Object-Oriented Programming (OOP)

## Applied Concepts
- Encapsulation
- Inheritance and polymorphism
- Separation of responsibilities between classes
- Collection manipulation

## Objective
To apply object-oriented programming concepts in the construction of an organized and functional system for library management.

## Execution
1. Download the file folder in .zip format
2. Extract the installed .zip file
3. Run Main.java

## Context
Academic project developed at IFRN in collaborative work.
